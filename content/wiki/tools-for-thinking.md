# tools for thinking

A structured reference companion to the journal entry [[tools for thinking|on tools for thinking]].

## writing instruments

**Index cards** — one idea per card, physical constraint enforces clarity.

**Freewriting** — write without stopping for a fixed duration, no editing. Surfaces what you actually think before the editor intervenes.

**The question journal** — instead of recording events or reflections, record only questions. Review monthly.

## conversational instruments

**Rubber duck debugging (applied generally)** — explaining a problem aloud to an imagined listener exposes gaps in your reasoning. The listener need not respond or even exist.

**Steelmanning** — before arguing against a position, construct the strongest possible version of it. If you can't, you don't understand it yet.

## structural instruments

**The pre-mortem** — before starting a project, imagine it has failed. Describe in detail why. Surfaces risks that optimism suppresses.

**Inversion** — ask what would guarantee failure, then avoid those things.
